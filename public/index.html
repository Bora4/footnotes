About this project

This project is still a work in progress

This project has been created to show my approach to backend architecture. All of the design choices belong to me. I have created layers of structures to support hypothetical future scaling. This project aims to support any type of frontend framework that can function with JSON responses from API endpoints. This project is not a showcase of my frontend skills. I have chosen typescript + node.js + express.js as my tech stack, simple html and css for frontend and, JSON files as data.

Important things I will address in future:

  • Security
  • Testing
  • Error handling
